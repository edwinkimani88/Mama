# Mama Network 🇰🇪
Where Hustle Meets Opportunity.

Mama Network is a simple, human-centered platform built for everyday Kenyans to find and access real work opportunities.

## Features
- **No CVs or Degrees:** Built for the real economy.
- **Tiered Fee System:** Workers pay a small fee to unlock contact details, scaled by job salary.
- **Fair & Fast:** Connecting "Wenye Kazi" directly with "Watafuta Kazi".
- **Trusted Profiles:** Verified workers with clear photos and track records.

## Fee Structure
- Under KSh 1,000 &rarr; KSh 100
- KSh 1,000 – 5,000 &rarr; KSh 200
- KSh 5,000 – 15,000 &rarr; KSh 1,000
- KSh 15,000+ &rarr; KSh 2,000

## How to use
Open `index.html` in any browser to get started.
